# OSCRaccoon by Recho Raccoon!
<img width="540" height="540" alt="OSCRaccoon Icon Transparent" src="https://github.com/user-attachments/assets/1d4b28e1-5c71-447d-9ce3-7c02a48e6a00" />

OSCRaccoon is an app for Quest standalone that can display the current song you're listening to, as well as custom cycling messages and your local time to your VRChat chatbox through OSC without a PC!

Download it [HERE](https://github.com/RechoRaccoon/OSC-Raccoon/releases/download/v1.1/OSCRaccoon.apk)!

OSCRaccoon falls under the GNU General Public License (GPL), meaning it's fully open source, but all forks are required to use this same license.
