# Standalone Chatbox tool for Quest VRChat - No PC Required!
- Display your music, time, custom Messages, and more in your VRChat chatbox VIA OSC!
- Supports ;ocal files and Spotify!
- Built in local file media-player!
- Organize your local files with customizable playlists! Image and video supported playlist covers!
- Create custom "Presets" to quickly swap between different vhatbox layouts and cycling message sets!
- 100% free and open-source!
Download the latest version [HERE](https://github.com/RechoRaccoon/OSC-Raccoon/releases/download/v2.3/Track.Raccoon.apk)!
